# Customer Churn Prediction

## Project Overview
This project predicts whether a telecom customer is likely to churn using Machine Learning.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-Learn
- Streamlit

## Model
- Logistic Regression
- Accuracy: 82%

## Features
- Customer Churn Prediction
- Interactive Web Application
- Streamlit Deployment
